## рҹҺҜ Problem Statement

The modern recruitment process is heavily guarded by Applicant Tracking Systems (ATS). Highly qualified candidates face rejection because:

* They don't understand how ATS algorithms weight specific keywords.
* They receive automated rejection emails with zero actionable feedback.
* They lack a structured roadmap to bridge the gap between their current skills and enterprise job requirements.

### рҹ’Ў Objective

To architect a **microservices-based AI application** that is:
* **Analytical:** Scoring resumes like a real enterprise ATS.
* **Constructive:** Providing brutal but honest feedback to fix weak points.
* **Scalable:** Decoupling the heavy ML tasks from the main API server.

Rizzume AI steps away from a traditional monolith and follows a decoupled **three-tier architecture**:

### 1пёҸвғЈ Machine Learning Layer (Python / FastAPI)

* Cross-service data streaming to the Python engine.

### 3пёҸвғЈ Client Layer (React.js)

* Secure file drag-and-drop zones.
* Interactive dashboards for ATS Score, Roast, and Roadmap.

## рҹ“Ҡ Core ML Logic: How the ATS Score Works

Unlike basic string-matching algorithms, Rizzume AI uses **Term Frequency-Inverse Document Frequency (TF-IDF)**. 

* **Weight Matters:** It understands that the word "Kubernetes" in a DevOps JD carries significantly more mathematical weight than the word "Teamwork".
* **Vector Comparison:** The system creates vectors for both the Resume and the Job Description, calculating the cosine similarity to generate a highly accurate, industry-standard percentage.

---

## рҹҡҖ Deployment & DevOps Engineering

**Overcoming Cloud Limitations:**
Both backend services are hosted on Render's free tier, which puts idle servers to sleep. To prevent `503 Timeout` errors during cross-service communication:
* Engineered custom `/health` routes on both Python and Node servers.
* Implemented specific schedule-based Cron-jobs to ping the central API during daytime hours, ensuring high availability without exhausting the 750-hour monthly cloud limit.
